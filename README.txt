ASTRALBOOKS VERSION 1.0.0 

INSTRUCTIONS:
1. Open up AstralBooks.jar
2. Set up chart of accounts by going to Edit -> Chart Of Accounts to add all the accounts needed
3. Set the initial balances of the balance sheet accounts in Edit -> Initial Balances
4. At this point, you can do File -> Save File to save all changes, and the zip file can be loaded
so that no changes are lost.
5. When transactiosn occur, they can be added in Edit -> New Transaction
6. Many different statements can be generated in the View menu after all the previous steps are
complete. An Excel file containing the corresponding statement will be generated in the specified
location.
